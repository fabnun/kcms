Documentación KCMS
=================
KCMS (Kreadi Content Management System) es un administrador de contenidos
Java para Google App Engine que usa BeansShell para simular JSP. Los contenidos
son almacenados en tablas que pueden contener subtablas.
Tipos de dato: String, Number, Boolean, Select, Script, Html, File, Id y SubTable.
 
JSP Simulado
----------------
cualquier archivo de tipo texto soporta scriptlets jsp del tipo
<%CODIGO JAVA(BEANSHELL);%> o <%=VALOR%>
 
Metodos Incluidos  en los scriptlets
-------------------------------------
getTable(String id) Obtiene una tabla
 
setTable(Table table) Guarda una tabla 
 
index() Obtiene la tabla asociada al id
 
log(Object s) genera una entrada en el log
 
include(String url) Incluye el texto del archivo o de un script procesado
 
append(Object s) Inserta un texto corresponde a out.write() de jsp
 
content(String url) Indica la url del script contenedor
 
include() Se usa dentro del script contendor para incluir el texto de subscript
